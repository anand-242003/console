# Community

KubeStellar Console is a sub-project of [KubeStellar](https://kubestellar.io) and participates in the broader KubeStellar community.

## Communication Channels

| Channel | Link | Purpose |
|---------|------|---------|
| **Slack** | [#kubestellar-dev](https://cloud-native.slack.com/channels/kubestellar-dev) on CNCF Slack | Development discussion, questions, help |
| **Mailing List** | [kubestellar-dev@googlegroups.com](https://groups.google.com/g/kubestellar-dev) | Announcements, governance votes, design proposals |
| **Private Mailing List** | [kubestellar-dev-private@googlegroups.com](https://groups.google.com/u/1/g/kubestellar-dev-private) | Security and Code of Conduct matters (maintainers only) |
| **Security** | [kubestellar-security-announce@googlegroups.com](mailto:kubestellar-security-announce@googlegroups.com) | Vulnerability reports (see [SECURITY.md](../SECURITY.md)) |
| **GitHub Issues** | [kubestellar/console/issues](https://github.com/kubestellar/console/issues) | Bug reports, feature requests |
| **GitHub Discussions** | [kubestellar/console/discussions](https://github.com/kubestellar/console/discussions) | Open-ended discussion, Q&A |

## Community Meetings

KubeStellar holds regular community meetings open to all participants:

- **KubeStellar Community Call** — Bi-weekly, covers all KubeStellar sub-projects including Console
  - Calendar: See [kubestellar.io/community](https://kubestellar.io/community) for schedule and join link
  - Agenda: Shared in advance on the mailing list; anyone can add items
  - Notes: Published after each meeting

### Proposing Agenda Items

To add a topic to an upcoming community call:

1. Post on [#kubestellar-dev Slack](https://cloud-native.slack.com/channels/kubestellar-dev), or
2. Email [kubestellar-dev@googlegroups.com](mailto:kubestellar-dev@googlegroups.com), or
3. Comment on the meeting agenda document (linked in the calendar invite)
